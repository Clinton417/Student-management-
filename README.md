# Student Management System

A complete local student management application built for XAMPP using PHP, MySQL, HTML5, CSS, JavaScript, and Bootstrap.

## Setup

1. Install XAMPP and start Apache and MySQL.
2. Copy this project to `C:\xampp\htdocs\student_management_system` (or your htdocs folder).
3. Open phpMyAdmin at `http://localhost/phpmyadmin`.
4. Create the database by running the SQL in `database.sql`.
5. Open `http://localhost/student_management_system` in your browser.

## Features

- Homepage with book animation.
- Login page with bubble animation.
- Admin dashboard with student management.
- Add, edit, delete student records.
- Responsive design for desktop and tablet.

## Database

The database name is `student_management`.

## Files

- `index.php` — Homepage
- `login.php` — Admin login page
- `dashboard.php` — Admin dashboard
- `add_student.php` — Add students
- `edit_student.php` — Edit students
- `delete_student.php` — Delete students
- `logout.php` — Admin logout
- `includes/db.php` — Database connection
- `css/style.css` — Application styles
- `js/script.js` — JavaScript interactions

## Notes

- Run this project locally with XAMPP on Windows.
- The login and admin flow are designed for development environments.
